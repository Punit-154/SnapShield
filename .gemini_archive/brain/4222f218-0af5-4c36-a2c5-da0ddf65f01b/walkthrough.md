# VIT Chennai PYQ Archive — Prediction Papers Walkthrough

## Summary
Analyzed **53 actual VIT Chennai question papers** and generated **24 predicted question papers** (3 per subject: CAT-1, CAT-2, FAT) across **8 subjects** for the 2026-2027 academic year.

## Analysis Methodology

### Papers Analyzed
Viewed actual VIT exam papers (scanned images + PDFs) to extract:
- **Exact exam format**: Header structure, table format with Q.No / Question / M / CO / BL columns
- **Mark distributions**: CAT = 5 × 10 = 50 marks (90 min), FAT = 10 × 10 = 100 marks (3 hrs)
- **Question styles**: Scenario-based, code analysis, prove/construct, compare, design
- **Topic frequency**: Which topics appear in every exam vs. rotating topics
- **Bloom's Level patterns**: CAT-1 tends BL2-BL3, CAT-2 goes BL3-BL4, FAT spans BL2-BL6

### Key Patterns Identified

| Pattern | Observation |
|---------|-------------|
| **Compiler Design** | Compiler phases question appears in EVERY paper. LALR(1) check is a FAT staple. Backpatching + SDT always tested. Loop optimization with matrix code is recurring. |
| **Cloud Architecture (AWS)** | 100% scenario-based. Every question presents a company problem requiring AWS architecture design. VPC/networking, IAM security, database migration, and cost optimization always appear. |
| **Smart Cities** | Q1 always split into conceptual (5M) + fill-in-blanks/True-False (5M). Sensor networks + transportation is a recurring topic. Case studies (Barcelona/Singapore/Dubai) expected. |
| **Cloud Computing** | MAX-MIN and SJF scheduling algorithms with VMs are tested numerically. Healthcare/enterprise migration scenarios are common. |
| **Discrete Mathematics** | Heaviest paper count (24 actual papers analyzed). Group theory proofs, graph coloring, recurrence relations, and Euler/Hamiltonian circuits dominate. |
