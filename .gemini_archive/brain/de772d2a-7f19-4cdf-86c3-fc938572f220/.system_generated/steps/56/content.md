A GNU GRUB-inspired developer portfolio built with Next.js and TypeScript.
Instead of scrolling through a traditional portfolio, visitors boot into an operating-system-inspired experience and navigate through projects, skills, experience, and contact information using a Linux-inspired interface.

AnsabOS is an experiment in creating a portfolio that feels more like software than a website.
Inspired by:
- GNU GRUB
- Linux terminals
- Package managers
- System logs
- Command-line workflows
Every page is designed to feel like a component of a running operating system.

### Boot Menu
Navigate the portfolio through a GNU GRUB-inspired bootloader.

### System Information
View developer information through a neofetch-style system overview.

### Package Manager Skills
Explore technologies as installed packages rather than traditional skill cards.

### Experience Logs
Read professional experience through a journalctl-inspired interface.

### Terminal Navigation
Interact with the portfolio using keyboard-driven workflows.

### GitHub Integration
Display live repository information, pinned projects, and development activity.

## Local Development
Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

Build for production:

```
npm run build
```
