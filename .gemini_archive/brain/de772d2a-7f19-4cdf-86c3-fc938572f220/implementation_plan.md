### 1.2 Router: 3-Layer Architecture (Research-Backed)

The 2026 consensus is **layered routing** — not a single classifier:

```
┌─────────────────────────────────────────────────────────┐
│                 Incoming Task                            │
└───────────────────────┬─────────────────────────────────┘
                        │
           ┌────────────▼────────────┐
           │  Layer 1: RULES ENGINE  │  ~0ms latency
           │  Keywords, length,      │  Deterministic
           │  privacy patterns       │
           └─────────┬──────────────┘
                     │ (no match)
           ┌─────────▼──────────────┐
           │  Layer 2: SEMANTIC     │  ~5-20ms latency
           │  ROUTER (embeddings)   │  Local, no LLM call
           │  aurelio-labs/          │  HuggingFace encoder
           │  semantic-router        │
           └─────────┬──────────────┘
                     │ (confidence < 0.7)
           ┌─────────▼──────────────┐
           │  Layer 3: LLM          │  ~100-300ms latency
           │  CLASSIFIER            │  Only for ambiguous
           │  (Ollama/phi4)          │  cases (~5% of traffic)
           └─────────┬──────────────┘
                     │
           ┌─────────▼──────────────┐
           │  PRIVACY OVERRIDE      │  Always checked
           │  Sensitive patterns →   │
           │  force Ollama local     │
           └─────────┬──────────────┘
                     │
           ┌─────────▼──────────────┐
           │  ROUTING TABLE         │  task_type → provider
           │  Select provider +     │
           │  model from config     │
           └─────────┬──────────────┘
                     │
           ┌─────────▼──────────────┐
           │  LiteLLM Proxy         │  Handles failover,
           │  (port 4000)           │  caching, cost tracking
           └────────────────────────┘
```

```python
# /opt/agentos/router/models.py
from pydantic import BaseModel
from enum import Enum

class TaskType(str, Enum):
    CODING_COMPLEX = "coding-complex"
    CODING_SIMPLE = "coding-simple"
    TESTING = "testing"
    RESEARCH = "research"
    PLANNING = "planning"
    QUICK_EDIT = "quick-edit"
    PRIVACY_SENSITIVE = "privacy-sensitive"

class ClassificationResult(BaseModel):
    task_type: TaskType
    complexity_score: int          # 1-10
    contains_sensitive_data: bool
    confidence: float              # 0.0-1.0
    reasoning: str                 # Why this classification
    classified_by: str             # "rules" | "semantic" | "llm"
    latency_ms: float              # Time to classify

class RoutingDecision(BaseModel):
    provider: str                  # "ollama" | "sambanova" | "copilot"
    model: str                     # e.g. "llama3.1:8b"
    classification: ClassificationResult
    latency_ms: float              # Total routing time
    cache_hit: bool = False        # Was this a cached decision?
```

```python
# /opt/agentos/router/rules.py
class RulesEngine:
    """Layer 1: Deterministic keyword + heuristic routing. ~0ms."""

    TESTING_KEYWORDS = {"test", "pytest", "jest", "unittest", "debug", "bug", "fix"}
    PRIVACY_KEYWORDS = {"password", "secret", "token", "api_key", ".env", "credential"}
    RESEARCH_KEYWORDS = {"research", "find", "search", "compare", "analyze", "explain"}
    CODING_KEYWORDS = {"implement", "write", "create", "build", "refactor", "function", "class"}

    def classify(self, task: str) -> ClassificationResult | None:
        """Returns classification if a rule matches, None otherwise."""
        words = set(task.lower().split())

        # Privacy always wins
        if words & self.PRIVACY_KEYWORDS:
            return ClassificationResult(
                task_type=TaskType.PRIVACY_SENSITIVE,
                complexity_score=5,
                contains_sensitive_data=True,
                confidence=1.0,
                reasoning=f"Matched privacy keywords: {words & self.PRIVACY_KEYWORDS}",
                classified_by="rules",
                latency_ms=0.0,
            )

        # Short queries = quick edit
        if len(task.split()) < 8:
            return ClassificationResult(
                task_type=TaskType.QUICK_EDIT,
                complexity_score=2,
                contains_sensitive_data=False,
                confidence=0.85,
                reasoning="Short query (<8 words)",
                classified_by="rules",
                latency_ms=0.0,
            )

        # Explicit testing keywords
        if words & self.TESTING_KEYWORDS:
            return ClassificationResult(
                task_type=TaskType.TESTING,
                complexity_score=5,
                contains_sensitive_data=False,
                confidence=0.9,
                reasoning=f"Matched testing keywords: {words & self.TESTING_KEYWORDS}",
                classified_by="rules",
                latency_ms=0.0,
            )

        return None  # No rule matched → pass to Layer 2
```

```python
# /opt/agentos/router/semantic.py
from semantic_router import Route
from semantic_router.layer import RouteLayer
from semantic_router.encoders import HuggingFaceEncoder  # Local, free

class SemanticClassifier:
    """Layer 2: Embedding-based intent routing. ~5-20ms, fully local."""

    def __init__(self):
        # Local encoder — no API calls, runs on CPU
        self.encoder = HuggingFaceEncoder(name="all-MiniLM-L6-v2")

        # Define routes with example utterances
        self.routes = [
            Route(name="coding-complex", utterances=[
                "implement a full REST API with auth and rate limiting",
                "refactor this module to use the strategy pattern",
                "build a React component with state management and API integration",
                "write a Rust macro that generates trait implementations",
                "create a WebAssembly module for real-time data processing",
            ]),
            Route(name="coding-simple", utterances=[
                "add a null check here",
                "rename this variable to something descriptive",
                "fix the indentation in this function",
                "add a docstring to this method",
                "change the return type to Optional",
            ]),
            Route(name="testing", utterances=[
                "write unit tests for the UserService class",
                "add integration tests for the API endpoints",
                "debug why this test is failing intermittently",
                "create a test fixture for database operations",
            ]),
            Route(name="research", utterances=[
                "compare React vs Svelte for this use case",
                "find the best caching strategy for this workload",
                "explain how WebAssembly handles memory safety",
                "analyze the performance of this algorithm",
            ]),
            Route(name="planning", utterances=[
                "plan the architecture for a microservices migration",
                "create a roadmap for the next sprint",
                "break this feature into smaller tasks",
                "design the database schema for user management",
            ]),
        ]

        self.layer = RouteLayer(encoder=self.encoder, routes=self.routes)

    def classify(self, task: str) -> ClassificationResult | None:
        """Returns classification if confidence > threshold."""
        start = time.monotonic()
        result = self.layer(task)
        elapsed = (time.monotonic() - start) * 1000

        if result.name is None:
            return None  # No confident match → pass to Layer 3

        # Map complexity by route type
        complexity = {"coding-complex": 8, "coding-simple": 3,
                      "testing": 5, "research": 6, "planning": 7}

        return ClassificationResult(
            task_type=TaskType(result.name),
            complexity_score=complexity.get(result.name, 5),
            contains_sensitive_data=False,
            confidence=result.confidence if hasattr(result, 'confidence') else 0.85,
            reasoning=f"Semantic match: {result.name} (similarity-based)",
            classified_by="semantic",
            latency_ms=elapsed,
        )
```

```python
# /opt/agentos/router/router.py
class AgentOSRouter:
    """3-layer router: Rules → Semantic → LLM → Privacy Override → Provider."""

    def __init__(self, config_path: str = "~/.agentos/config.yaml"):
        self.config = load_config(config_path)
        self.rules = RulesEngine()
        self.semantic = SemanticClassifier()
        self.llm = LLMClassifier(self.config["router"])
        self.logger = RoutingLogger(self.config["logging"])
        self._cache: dict[str, RoutingDecision] = {}  # Simple LRU cache

    async def route(self, task: str) -> RoutingDecision:
        """Classify task through 3 layers and select provider."""
        start = time.monotonic()

        # Check cache first (exact match)
        cache_key = hashlib.md5(task.encode()).hexdigest()
        if cache_key in self._cache:
            cached = self._cache[cache_key]
            cached.cache_hit = True
            return cached

        # Layer 1: Rules (0ms)
        result = self.rules.classify(task)

        # Layer 2: Semantic Router (5-20ms)
        if result is None:
            result = self.semantic.classify(task)

        # Layer 3: LLM Classifier (100-300ms, only ~5% of traffic)
        if result is None:
            result = await self.llm.classify(task)

        # Privacy override: always checked regardless of layer
        if self._contains_sensitive_data(task):
            result.contains_sensitive_data = True
            result.task_type = TaskType.PRIVACY_SENSITIVE

        # Select provider from routing table
        provider_name = self.config["router"]["routes"][result.task_type.value]
        provider = self.config["providers"][provider_name]

        elapsed = (time.monotonic() - start) * 1000

        decision = RoutingDecision(
            provider=provider_name,
            model=provider["default_model"],
            classification=result,
            latency_ms=elapsed,
        )

        # Cache + log
        self._cache[cache_key] = decision
        await self.logger.log(decision)
        return decision

    def _contains_sensitive_data(self, task: str) -> bool:
        """Check for privacy-sensitive patterns."""
        patterns = self.config["privacy"]["sensitive_patterns"]
        return any(p in task.lower() for p in patterns)

```

```
   - python3, python3-venv, python3-pip
   - git, curl, wget, jq, build-essential
   - Node.js 22 (for n8n)
   - Docker Engine (apt, not Docker Desktop)

   curl -fsSL https://ollama.com/install.sh | sh

   curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
5. Clone AgentOS repo to /opt/agentos
6. Create Python venv, install router deps:
   pip install litellm pydantic pyyaml aiosqlite
7. Install n8n globally:
   npm install -g n8n

   - ollama.service
   - litellm-proxy.service (port 4000)

   ln -s /opt/agentos/skills/* ~/.hermes/skills/

    cp /opt/agentos/config/defaults.yaml ~/.agentos/config.yaml

    ~/.agentos/.env (with placeholders for API keys)
12. Run first-time setup:
    hermes --skill agentos-setup
13. Health check:
    /opt/agentos/scripts/health-check.sh

Idempotency:
  - Every step checks if already done before executing
  - Re-running is safe
  - Uses markers in ~/.agentos/.installed
```

## 6. Security Model

| Layer | Measure |
|---|---|
| **API Keys** | Stored in `~/.agentos/.env`, loaded via `${VAR}` in YAML, never in git |
| **User Isolation** | Hermes runs as non-root `agentos` user inside WSL |
| **Code Execution** | Optional Docker sandbox via `hermes config set terminal.backend docker` |
| **Privacy** | Sensitive patterns auto-detected → forced to local Ollama |
| **Email** | Gmail App Passwords (not main password), IMAP over TLS |
| **n8n** | Localhost-only binding (not exposed to network) |
| **Host Isolation** | Separate WSL distro — won't affect your Arch/Kali setups |
