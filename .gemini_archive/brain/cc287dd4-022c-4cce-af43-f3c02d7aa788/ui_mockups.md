# Dashboard UI/UX Mockups — Client Review

> **Client Profile**: Professional woman, age 40-50  
> **Requirements**: White + Purple + Yellow, easy on the eyes, easy to navigate  
> **Design Principles**: Large text (18px+), generous spacing, rounded corners, minimal clutter

---

## Option A — Sidebar Dashboard

![Sidebar dashboard with purple navigation, stat cards, and quick generate section](C:/Users/joel0/.gemini/antigravity/brain/cc287dd4-022c-4cce-af43-f3c02d7aa788/mockup_sidebar_dashboard_1781448226042.png)

**Layout**: Left sidebar navigation + main content area  
**Best for**: Power users who run the pipeline frequently  
**Navigation**: Always-visible sidebar with Dashboard, Generate, History, Settings  
**Key features**:
- 4 stat cards at top (Total Listings, Pass Rate, Avg Time, Active Models)
- Quick Generate input prominently placed
- Recent runs table with status badges

---

## Option B — Top Navigation + Cards

![Top navigation with welcoming hero section and 3 info cards](C:/Users/joel0/.gemini/antigravity/brain/cc287dd4-022c-4cce-af43-f3c02d7aa788/mockup_topnav_cards_1781448257339.png)

**Layout**: Horizontal top nav + card-based content  
**Best for**: Clients who prefer a clean, welcoming experience  
**Navigation**: Simple tab bar (Home, Generate, My Listings, Reports, Settings)  
**Key features**:
- Personalized greeting ("Good Evening!")
- Large, prominent search/input for niche
- 3 card panels: Recent Listings, Performance, Quick Actions

---

## Option C — Minimal Centered (Description)

**Layout**: Single-column, centered, zen-like  
**Best for**: Maximum simplicity, feels like Apple design  
**Navigation**: Minimal — just a header and one main action  
**Key features**:
- "What would you like to create today?" welcome prompt
- Single large input field front and center
- Activity timeline below showing recent generations
- Minimal stats bar at bottom

---

## Color Palette (All Options)

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| **Background** | White | `#FFFFFF` | Main background |
| **Surface** | Soft Lavender | `#F5F3FF` | Cards, panels |
| **Primary** | Purple | `#7C3AED` | Navigation, buttons, headings |
| **Primary Light** | Light Purple | `#DDD6FE` | Hover states, borders |
| **Accent** | Warm Yellow | `#F59E0B` | CTAs, success badges, highlights |
| **Accent Light** | Soft Gold | `#FEF3C7` | Warning backgrounds |
| **Text** | Charcoal | `#1F2937` | Body text |
| **Muted** | Gray | `#6B7280` | Secondary text, timestamps |

## Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Headings | Inter | 24-32px | 600 |
| Body | Inter | 16-18px | 400 |
| Labels | Inter | 14px | 500 |
| Stats | Inter | 36-48px | 700 |
